# Для кого он

Это курсовой проект на тему "Разработка мобильного приложения судоку с мультиплеером". Здесь есть как серверная, так и клиентская(пока в терминале) части. 

# Как использовать

Неоходимо зайти в директорию `server` и запустить командой
```bash
go build && ./server &
```

Для подключения надо зайти в директорию `client` и запустить командой
```bash
go build && ./client
```
Изнчально требуется чтобы каждый из 2 игроков запустил `client`.
